# Beacon Demo — Voiceover Script

Target: 60-80s. 6 clips. Stellar Hacks judges.
Framework: PAS (show product first, then explain).
Audio: ElevenLabs Brian, stability 0.82, style 0.03.

## 01-dashboard
**Frame:** Full Pulse dashboard — gauge at top, featured endpoint cards with full-width probe bars
**Text:** "Eight endpoints. Probed every five minutes. Scored on uptime and latency. This is Beacon's live trust dashboard for x402 on Stellar."

## 02-cards
**Frame:** Close-up of featured endpoint cards — probe bars spanning full width, trust scores, uptime inline
**Text:** "Each endpoint gets a trust score from zero to one hundred. The bars are real probe results. Green is a successful response. Red is a failure or timeout. The score combines uptime and latency over twenty-four hours."

## 03-compact
**Frame:** Compact rows (dead endpoints dimmed) + feed section with timestamps and latencies
**Text:** "Endpoints that aren't responding drop to zero and fade out. The probe log below shows every request with its actual latency. No fake data. Every number is a real HTTP call."

## 04-payment
**Frame:** Terminal showing test-client making x402 payment and getting trust score JSON back
**Text:** "The scores are behind an x402 paywall. An agent pays one tenth of a cent in USDC on Stellar. It gets the trust score JSON back. That transaction settles on Stellar testnet."

## 05-explorer
**Frame:** Stellar testnet explorer showing account, trustline, DEX swap, and payment transactions
**Text:** "Here's the proof on chain. Account creation. USDC trustline. A DEX swap for testnet USDC. And the x402 payment. All real transactions on Stellar."

## 06-close
**Frame:** Branded closing card — Beacon logo, tagline, live URL, GitHub
**Text:** "Beacon gives agents trust data before they spend money. Try it live."
